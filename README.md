# Random Task Generator

**Автор:** Габбазов Алмаз

**Назначение:** генератор случайных задач для итоговой аттестации по курсу «Код будущего».

## Описание

Программа помогает принимать решения при выборе деятельности — генерирует случайные задачи из списка с возможностью фильтрации по категориям и сохранением истории.

## Функциональность

* **Генерация случайных задач** — выбор случайной задачи из общего списка или отфильтрованного по категории.
* **Добавление пользовательских задач** — возможность добавлять свои задачи с присвоением категории.
* **Фильтрация задач** — отбор задач по категориям («Учёба», «Спорт», «Работа»).
* **История генераций** — отображение всех сгенерированных задач в таблице с датой и временем.
* **Сохранение истории** — автоматическое сохранение в JSON‑файл между запусками.

## Инструкция по запуску

### Требования

* Python 3.6 или выше;
* стандартная библиотека `tkinter` (включена в Python).

### Шаги по запуску

1. Убедитесь, что Python установлен и доступен в командной строке.
2. Скачайте файл `RandomTaskGenerator.py` на ваш компьютер.
3. Откройте командную строку (терминал) и перейдите в папку с файлом:
