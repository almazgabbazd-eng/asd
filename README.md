# Random Task Generator

**Автор:** 

## Описание

Приложение для генерации случайных задач с фильтрацией по типам и сохранением истории.

## Функциональность

* генерация случайных задач из предопределённого списка;
* добавление пользовательских задач;
* фильтрация задач по типам (Учёба, Спорт, Работа);
* просмотр истории сгенерированных задач;
* очистка истории;
* сохранение истории между запусками.

## Инструкция по запуску

### Требования

* Python 3.6 или выше;
* стандартная библиотека `tkinter` (включена в Python).
