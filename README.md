# Random Task Generator

**Автор:** [Ваше Имя Фамилия]

## Описание

Приложение для генерации случайных задач с фильтрацией по типам и сохранением истории. Позволяет:
* генерировать случайные задачи из предопределённого списка;
* добавлять собственные задачи;
* фильтровать задачи по типам (Учёба, Спорт, Работа);
* просматривать историю сгенерированных задач;
* очищать историю;
* сохранять историю между запусками приложения.

## Функциональность

* **Генерация случайных задач** — кнопка «Сгенерировать случайную задачу» выбирает случайную задачу из списка с учётом фильтра.
* **Добавление пользовательских задач** — возможность добавлять свои задачи с выбором типа.
* **Фильтрация** — фильтр по типу задачи (Все, Учёба, Спорт, Работа) применяется как к генерации, так и к отображению истории.
* **История** — отображение всех сгенерированных задач с датой и временем в виде таблицы.
* **Сохранение данных** — история сохраняется в JSON‑файл между запусками.
* **Валидация ввода** — проверка на пустые задачи при добавлении.
* **Обработка ошибок** — уведомления об ошибках сохранения/загрузки истории.
